# API REST TERAPIFY

Proyecto generado principalmente con NodeJS y Typescript.

## Instalación

Ejecutar la siguiente linea:

```

npm install

```

Ejecutar pruebas unitarias

```

npm run test

```

Entorno de desarrollo.
Ejecutar la siguiente linea:

```

npm run dev

```

Entorno de producción
Ejecutar la siguiente linea:

```

npm run start

```

### Tecnología usada
